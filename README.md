# How to Install
1: Download and extract the repository as a .zip file.

2: Open your browser and navigate to Settings → Extensions.

3: Enable Developer Mode.

4: Click "Load unpacked" and select the root folder of yt-auto-skip-ads.

5: You should now see YT Auto Skip Ads listed among your extensions.

6: (Optional) Disable Developer Mode after installation for added safety.
