# Yt-Auto-Skip-Ads
This extension, created by Darvin Dobariya, is intended for testing and fun purposes only. YT Auto Skip Ads may enable automatic skipping of YouTube ads, but it is not designed for commercial use or distribution.
Use this extension at your own risk. The creator assumes no responsibility for any legal actions, account suspension, or bans resulting from its use.

# How to Install This Extension
Download .zip file of this repository, and extract it.

Step 1: Open your browser and go to Settings → Extensions.

Step 2: Enable Developer Mode.

Step 3: Click on "Load unpacked" and select the root folder of yt-auto-skip-ads.

Step 4: You should now see YT Auto Skip Ads listed among your extensions.

Step 5: (Optional) Disable Developer Mode after installation for added safety.

# Note!
This extension has only been tested on Google Chrome and Microsoft Edge.
It may not work on other browsers.
